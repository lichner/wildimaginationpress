# Wild Imagination Press

Jekyll website for Wild Imagination Press KDP book publishing.

## Quick Start

```bash
# Install dependencies
bundle install

# Run dev server
bundle exec jekyll serve

# Build for production
bundle exec jekyll build
```

## Adding Books

Create `.html` file in `_books/` with front matter. See `WARP.md` for details.

## Deployment

Auto-deploys to GitHub Pages on push to main.
